# Stop Motor Controller

Stops a Motor393 connected to a Motor Controller 29.

```python
motor393.stop()
```

## How To Use

The example below will stop a Motor393 device after spinning for 3 seconds.

```python
motor393.spin(FORWARD)
wait(3, SECONDS)
motor393.stop()
```
