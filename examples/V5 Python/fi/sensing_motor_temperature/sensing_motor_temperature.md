# Moottorin lämpötila

Raportoi V5 Älymoottorin lämpötilan.

```don
motor.temperature(PERCENT)
```

## Miten käytetään

V5 Älymoottorin lämpötila -komennon avulla voi estää moottorin kuumenemisen raskaan ajon aikana.

Lämpötila raportoidaan välillä **0% - 100%**.
