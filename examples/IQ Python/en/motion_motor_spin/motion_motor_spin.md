# Spin 

Spins an IQ Motor or Motor Group indefinitely.

```python
motor.spin(DIRECTION)
```

## How To Use

Choose which **DIRECTION** the Motor or Motor Group will spin to with either **FORWARD** or **REVERSE** as the parameter.

```python
motor.spin(FORWARD)
```
