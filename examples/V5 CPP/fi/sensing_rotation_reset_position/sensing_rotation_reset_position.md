# Rotation.resetPosition();

Resetoi V5 Rotation Sensorin asentoarvon arvoon 0.

```cpp
Rotation.resetPosition();
```

## Miten käytetään

`Rotation.resetPosition()` voi resetoida Rotation Sensorin asennon arvoon 0.

```cpp
Rotation.resetPosition();
```
