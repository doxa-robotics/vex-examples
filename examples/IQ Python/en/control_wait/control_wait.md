# Wait

Waits for a specific amount of time before moving to the next command.

```python
wait(TIME, UNITS)
```

## How To Use

Set an amount of time in **SECONDS** or **MSEC** on the **UNITS** parameter to have your program **wait** before executing subsequent commands.

```python
wait(10, SECONDS)
```
