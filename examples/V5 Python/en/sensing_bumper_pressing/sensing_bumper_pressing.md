# Pressing Bumper
 
Reports if the Bumper Switch is currently being pressed.

```don
bumper.pressing()
```

## How To Use

Pressing bumper reports **True** if the Bumper Switch is being pressed.

Pressing bumper reports **False** if the Bumper Switch is not being pressed.
