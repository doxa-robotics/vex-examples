# Brain Timer Clear

Resets the V5 Brain's timer.

```cpp
Brain.Timer.clear();
```

## How To Use

The Brain's timer begins at the beginning of each project. `Brain.Timer.clear()` is used to reset the timer back to 0 seconds.
