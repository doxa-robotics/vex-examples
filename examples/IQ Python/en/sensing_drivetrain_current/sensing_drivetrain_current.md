# Drive Current

Reports the amount of current that the Drivetrain is currently using.

```python
drivetrain.current(UNITS)
```

## How To Use

`Drive Current` reports a range from **0.0 to 2.5** when **CurrentUnits.AMP** is passed as the **UNITS** parameter.
