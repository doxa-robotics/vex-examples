# Pressing Bumper
 
Reports if a VEX IQ Bumper is currently being pressed.

```python
bumper.pressing()
```

## How To Use

`Pressing Bumper` reports **True** if the Bumper is being pressed.

`Pressing Bumper` reports **False** if the Bumper is not being pressed.
