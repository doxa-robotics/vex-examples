# Magnet Set Power

Sets the power of a Electromagnet that will be used when picking up a magnetic object.

```cpp
Magnet.setPower(power);
```

## How To Use

The `Magnet.setPower()` command accepts a range of values from **0 to 100**  as a parameter.

```cpp
Magnet.setPower(100);
Magnet.pickup();
```
