# Kiertokulma

Antaa Rotation Sensorin mittaaman kiertokulman arvon.

```don
rotation.angle()
```

## Miten käytetän

`rotation.angle()` antaa decimaaliluvun, jonka Rotation Sensori on mitannut asteina.
