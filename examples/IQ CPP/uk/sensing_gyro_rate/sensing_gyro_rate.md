# Gyro Rate

Передає кутову швидкість гіроскопічного датчика VEX IQ.

```cpp
Gyro.rate()
```

## Як це працює

`Gyro.rate` передає значення в діапазоні **від 0 до 249.99** градусів за секунду (DPS).
