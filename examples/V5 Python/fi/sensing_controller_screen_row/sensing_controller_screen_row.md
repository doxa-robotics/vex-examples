category: sensing

# Kursorin rivi

Antaa ohjaimen näytön kursorin rivin.

```don
controller.screen.row()
```

## Miten käytetään

Kusorin rivi voi olla välillä **1 - 3**.

V5 ohjaimen näytöllä fonttia ei voi muuttaa.

**V5 Ohjain Fontti Koko - Rivimäärä:**

* Standardi Fontti - 3 Riviä (Oletus)
