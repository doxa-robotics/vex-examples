# GPS Calibrate

Starts the calibration process for a GPS Sensor.

```python
gps.calibrate()
```

## How To Use

A GPS Sensor must remain still during the calibration process.

```python
# Calibrate the GPS Sensor before starting any program code
gps.calibrate()
```
