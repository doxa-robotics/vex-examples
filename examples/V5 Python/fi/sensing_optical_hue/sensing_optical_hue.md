# Optical värisävy

Raportoi esineen värisävyn.

```don
optical.hue()
```

## Miten käytetään

`optical.hue()` antaa numeroarvon kohteen värille. 

Arvot ovat välillä **0** -**359**, ja se on värin paikka Vex värirenkaassa (värispetrissä).
