# Ajopelin vetomoottoreiden lämpötila
 
Antaa ajopelin V5 Älymoottoreiden lämpötilan.

```don
drivetrain.temperature(PERCENT)
```

## Miten käytetään

Ajoplein ajomoottoreriden lämpötila mitataan välillä **0% - 100%**.
