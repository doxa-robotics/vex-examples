# Kohteen nopeus

Antaa kavaitun kohteen nopeuden yksikkönä m/s.

```
distance.object_velocity()
```
## Miten käytetään

**Object velocity** raportoi kohteen nykyisen nopeuden m/s (metriä per sekunti).
