# Object Distance

Reports the distance of the nearest object from the Distance sensor.

```don
distance.object_distance(MM)
```

## How To Use

**Object distance** reports a range from **20mm to 2000mm**.

Acceptable values for `UNIT` are: MM or INCHES.
