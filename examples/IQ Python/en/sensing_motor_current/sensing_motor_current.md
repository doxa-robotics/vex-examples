# Motor Current

Reports the amount of current an IQ Motor or Motor Group is using.

```python
motor.current(UNITS)
```

## How To Use

`Motor Current` reports a range from **0.0 to 2.5** when **CurrentUnits.AMP** is passed as the **UNITS** parameter.
