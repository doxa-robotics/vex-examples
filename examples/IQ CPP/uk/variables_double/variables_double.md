# Double Variable

Встановлює змінну типу `double`.

`double myDouble = 1.5;`

## Як це працює

Змінна типу `double` - це число з плаваючою комою (десяткове).

`double pi = 3.14;`

Створіть унікальне і характерне ім'я. Потім, присвойте значення змінній.

`double turnAngle = 40.5;`
