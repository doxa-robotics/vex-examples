# Ohjainvärinä

Laittaa V5 ohjaimen värisemään.

```python
controller.rumble(".-.-")
```

## Miten käytetään

`controller.rumble(".-.-")` komento käyttää V5 ohjaimen värinämoottoria.

Voit säätää värinää "." ja "-" merkeillä. Merkki "." luo lyhyen ja merkki "-" pitkän värinän.
