# LED Päällä/Pois päältä

Laittaa LED päälle ja pois päältä.

```don
led.on()
```
```don
led.off()
```

## Miten käytetääm

`led.on()` laittaa LED lampun päälle.

`led.off()` laittaa LED lampun pois päältä.
