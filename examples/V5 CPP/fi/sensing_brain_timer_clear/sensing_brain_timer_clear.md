# Nollaa kello

Nollaa V5 Aivojen kellon.

```cpp
Brain.Timer.clear();
```

## Miten käytetään

Aivojen kello käynnistyy ohjelman suorituksen alussa. Nollaa kello -komentoa voi käytttää ohjelman aikana aloiottaamaan ajanlasku uudelleen arvosta 0 sekuntia.
