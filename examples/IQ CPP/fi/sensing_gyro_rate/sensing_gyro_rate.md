# Gyro Nopeus

Raportoi VEX IQ Gyro sensorin kulmanopeuden.

```cpp
Gyro.rate()
```

## Miten käytetääm

`Gyro.rate` antaa arvon väliltä **0 to 249.99** yksikkönä astetta per sekunti (DPS).
