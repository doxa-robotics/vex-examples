# Optical kirkkaus

Antaa valonmäärän, jonka Optical sensori havaitsee.

```don
optical.brightness()
```

## Miten käytetään

`optical.brightness()` antaa arvot välillä **0%** - **100%**

Paljon valoa antaa korkean arvon.

Vähän valoa antaa pienen arvon.
