# Double Variable

Sets a variable to a `double` type.

`double myDouble = 1.5;`

## How To Use

The `double` variable type is a number that has a decimal point.

`double pi = 3.14;`

Create a unique and descriptive variable name. Then, assign a value to your variable.

`double turnAngle = 40.5;`
